## Prerequisites Checklist

Before starting, please ensure you have completed the following:

- [ ] **Git installed** (latest stable version) - Verify with: `git --version`
- [ ] **GitHub account created** (free account at github.com)
- [ ] **GitHub Desktop installed** (optional but recommended)
- [ ] **Code editor installed** (VS Code recommended)
- [ ] **Command line access** (Terminal on Mac/Linux, Git Bash on Windows)
- [ ] **Git configured** with your name and email:
  ```bash
  git config --global user.name "Your Name"
  git config --global user.email "your.email@example.com"
  ```

## Installing Markdown Preview Enhanced in Cursor

To install the Markdown Preview Enhanced plugin in Cursor, follow these steps:

### Method 1: Using the Extensions Panel (Recommended)

1. **Open the Extensions panel:**

   - Press `Cmd+Shift+X` (Mac) or `Ctrl+Shift+X` (Windows/Linux)
   - Or click the Extensions icon in the left sidebar (square icon with four squares)

2. **Search for the extension:**

   - In the search box at the top, type: `Markdown Preview Enhanced`
   - Look for the extension by **Yiyi Wang** (shuenhoy)

3. **Install the extension:**

   - Click the **Install** button on the extension card
   - Wait for the installation to complete

4. **Verify installation:**
   - After installation, you should see an "Installed" label on the extension
   - You may need to reload Cursor for the extension to be fully activated

### Method 2: Using Command Palette

1. **Open Command Palette:**

   - Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)

2. **Install extension:**
   - Type: `Extensions: Install Extensions`
   - Press Enter
   - Search for `Markdown Preview Enhanced`
   - Click Install

### Using Markdown Preview Enhanced

Once installed, you can use Markdown Preview Enhanced by:

- **Opening a Markdown file** (`.md` extension)
- **Right-click** in the editor and select `Markdown Preview Enhanced: Open Preview to the Side`
- Or use the keyboard shortcut: `Cmd+K V` (Mac) or `Ctrl+K V` (Windows/Linux)
